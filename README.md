# 🌍 FIFA World Cup 2026 - Round 1 Player Analysis

This project analyzes player statistics from Round 1 of the FIFA World Cup 2026 using Python, pandas, and matplotlib.  
It includes **data cleaning, player-level insights, team comparisons, and visualizations** to highlight key performances.

## 📂 Project Files
- `WorldCup2026_Analysis.ipynb` → Jupyter Notebook with full analysis workflow.
- `players.xlsx` → Dataset containing player statistics.

## 📊 Tasks Covered
- League Representation
- Top Scorers & Playmakers
- Discipline Analysis
- Goalkeeper Performance
- Age & Experience Distribution
- Minutes & Utilization

## 🎯 Key Insights
- Lionel Messi → Top scorer (3 goals in 79 minutes).
- Emiliano Martínez → Best GK (100% clean sheet).
- Nestory Irankunda → Youngest player (~19 years).
- Edin Džeko → Oldest player (~40 years).
- Algeria → Most aggressive team.
- Argentina → Most disciplined squad.
